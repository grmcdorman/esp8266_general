Detailed Description

An info setting is a read-only string setting.

On the UI, it is presented with the value simply output as-is (no HTML encoding). The value will be updated when values are fetched from the server.

The Web Server provides a mechanism for periodic updates; in JavaScript, adding the tab name, or the tab name, an ampersand, and the string "setting=" followed by the setting name to the array periodicUpdateList will result in the value being fetched approximately every 5 seconds. For example, if the tab is "Overview" and the setting is "uptime", then:

  • window.addEventListener(\"load\\", () => { periodicUpdateList.push("Overview"); }); will update all fields on the tab periodically;
  • window.addEventListener(\"load\\", () => { periodicUpdateList.push("Overview&setting=uptime"); }); will fetch and update the "uptime" field periodically.

Avoid updating an entire tab if it contains input fields; the update will refresh the input fields as well as the other fields.

The value will also be requested on initial page load, and when the user clicks "Reset Form".
