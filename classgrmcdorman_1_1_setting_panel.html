Detailed Description

The Setting Panel is the controller for a set of Setting.

It manages reading the values in from a POST request, and constructing the output JSON for the values when requested by the UI.

◆ on_post()

void grmcdorman::SettingPanel::on_post ( AsyncWebServerRequest *  request)

Handle a POST set-values request.

All settings that are updated from the incoming request; if a setting does not appear, it is updated with its default (set_default()) if it is a setting that is sent on requests (send_to_ui() is true).

Passwords are the only present case where send_to_ui() is false, meaning they are only set if explicitly included in the POST request.

Note that some settings, notably note and info settings, ignore any attempt to set a value from this call.
