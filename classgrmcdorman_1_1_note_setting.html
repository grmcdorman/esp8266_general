Detailed Description

A note setting.

A note is a read-only setting that spans both columns of the setting table. The text of the note is not escaped and can contain HTML code, including Javascript. Because it is read-only, no container name or setting name is used, and no identifier is set into the constructed HTML.

The note text can be changed after construction; this may not have any effect on the web page.
